# 🩺 Health AI Assistant

A healthcare assistant built with Python, Streamlit, and Groq LLM that allows users to upload medical reports, analyze laboratory values, ask health-related questions, receive hospital suggestions, and book appointments through a conversational interface.

## Features

### 📄 Medical Report Upload

* Upload PDF medical reports.
* Extract text using `pypdf`.
* Parse laboratory values using regular expressions.

### 🧪 Lab Value Analysis

* Compare extracted values with reference ranges.
* Classify values as:

  * Low
  * Normal
  * High
* Calculate severity for abnormal values.

### 📊 Risk Score and Triage

* Generate a risk score from laboratory abnormalities.
* Assign a triage category:

  * ✅ Safe
  * ⚠️ Moderate
  * 🚨 Emergency

### 🤖 Medical Chat Assistant

* Powered by Groq LLM (`llama-3.1-8b-instant`).
* Answers health-related questions.
* Uses:

  * Uploaded medical report
  * Hospital database
  * Recent conversation history

### 🏥 Hospital Recommendation

* Suggest hospitals from a predefined JSON database.
* Display hospital ratings and specialties.

### 📅 Appointment Booking

* Book appointments through natural language.
* Cancel appointments.
* Store appointments during the active session.

## Installation

Clone the repository:

```bash
git clone https://github.com/siyachopra768/health-ai-agent.git
cd health-ai-agent
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## Environment Variables

Create a `.env` file:

```env
GROQ_API_KEY=your_groq_api_key
```

---

## Run the Application

```bash
streamlit run app.py
```
